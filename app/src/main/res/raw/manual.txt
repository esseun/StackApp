StackApp User ManualPush Function -Input number on the top of the stack:1. Click on the edit text field. 2. Key in a number from 0 to 9.3. Click on Push button.For the push function, the system displays an error message while the stack is full(3 numbers in the stack) or if input type is out of defined range.Push Button Scenario:1. Click on the edit text field.2. Click on 5 button on number pad.3. Click on Push button.4. The application returns: 5 is pushed. Stack: [5]5. Click on the edit text field. 6. Click on 7 button on number pad.7. Click on Push button.8. The application returns: 7 is pushed. Stack: [5 7]	Pop Function -Pop out the number on the top of the stack:1. Click on the Pop button.For the pop function, the system displays error messages while the stack is empty.Pop Button Scenario:	(Continue on Push Button Scenario)1. The system returns: 7 is pushed. Stack: [5 7]2. Click on Pop button.3. The system returns: Popped: 7. Stack: [5]4. Click on Pop button.5. The system returns: Popped: 5. Stack: Empty. Quit Function -Terminate the Stack application and return to the desktop.	1. Click on the Quit button.
